          Configure Neovim Colorscheme

          Friday, 14 October 2022
          Neovim, Config, Customize, Terminal, Text Editor
          neovim, open source, text editor, vim

          In this case, I use Material theme.

          Installation #

          Install using packer (place in ~/.config/nvim/lua/packer-config/init.lua):

          use 'marko-cerovac/material.nvim'

          Configuration #

          Create config file ~/.config/nvim/colors-config/material.lua:

          ~/.config/nvim/lua/colors-config/material.lua

          require('material').setup({
  contrast = {
    sidebars = true, -- Enable contrast for sidebar-like windows ( for example Nvim-Tree )
    line_numbers = true, -- Enable contrast background for line numbers
    sign_column = true, -- Enable contrast background for the sign column
    cursor_line = true, -- Enable darker background for the cursor line
    non_current_windows = true, -- Enable darker background for non-current windows
  },
  italics = {
    comments = true, -- Enable italic comments
    keywords = true, -- Enable italic keywords
    functions = true, -- Enable italic functions
  },
})

vim.g.material_style = "deep ocean"
vim.cmd 'colorscheme material'

          Require config in ~/.config/nvim/init.lua.

          require('colors-config.material')
