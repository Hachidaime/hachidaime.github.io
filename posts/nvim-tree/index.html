          Nvim Tree #

          nvim-tree is a file explorer tree for neovim written in lua.

          Installation #

          nvim-web-devicons is optional and used to display file icons. It requires a patched font.

          Install using packer (place in ~/.config/nvim/lua/packer-config/init.lua):

          use 'kyazdani42/nvim-web-devicons'
use 'kyazdani42/nvim-tree.lua'

          Configuration #

          Create config file ~/.config/nvim/nvim-tree-config/init.lua:

          ~/.config/nvim/lua/nvim-tree-config/init.lua
