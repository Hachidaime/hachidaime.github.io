    Concenate Files using FFMPEG

    10 October 2022
    Linux, Terminal, Multimedia, FFMPEG
    linux, open source, multimedia, ffmpeg

    Example #

    1. Create file list
    echo file file1.mp4 >  mylist.txt
echo file file2.mp4 >> mylist.txt
echo file file3.mp4 >> mylist.txt
    1. Concenate Files
    ffmpeg -f concat -i mylist.txt -c copy output.mp4
