          Customize Neovim Start Screen with Dashboard-nvim

          Saturday, 15 October 2022
          Neovim, Config, Customize, Terminal, Text Editor
          neovim, open source, text editor, vim

          dashboard-nvim is a fancy fastest async start screen plugins for Neovim.

          Installation #

          Install using packer (place in ~/.config/nvim/lua/packer-config/init.lua):

          use 'glepnir/dashboard-nvim'

          Configuration #

          Create config file ~/.config/nvim/dashboard-config/init.lua:

          ~/.config/nvim/lua/dashboard-config/init.lua
