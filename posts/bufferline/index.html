          Manage Neovim Buffers with Bufferline

          Friday, 14 October 2022
          Neovim, Config, Customize, Terminal, Text Editor
          neovim, open source, text editor, vim

          Bufferline is buffer line (with tabpage integration) for Neovim built using lua.

          Installation #

          Install using packer (place in ~/.config/nvim/lua/packer-config/init.lua):

          use 'akinsho/bufferline.nvim'

          Configuration #

          Create config file ~/.config/nvim/bufferline-config/init.lua:

          ~/.config/nvim/lua/bufferline-config/init.lua

          require("bufferline").setup {
  options = {
    modified_icon = '●',
    close_icon = '',
    separator_style = 'thin',
    indicator = {
      icon = '█ ',
      style = 'icon'
    }
  },
}

          Require config in ~/.config/nvim/init.lua.

          require('bufferline-config')
